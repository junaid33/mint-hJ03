---
title: "Koala"
---

Add the following to your `mint.json` file to send analytics to Koala.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"analytics": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
